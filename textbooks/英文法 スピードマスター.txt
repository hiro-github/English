Lesson1 動詞・文型
owe A B , owe B to A (ex The firm owes its current success to the strong economy.)
  : BはAのおかげ(せい)である
rise,arise 自動詞
raise 他動詞
charge A for B
  : AにBを請求する
save A B (ex: save the employees a lot of time)
  : AからBを取り除く
lie- lay-lain- lying  自動詞
lay-laid-laid-laying 他動詞
tell O to do
  : Oにdoするように言う
have an accident
  : 事故を起こす
lend A B
  : AにBを貸す

  : ちょうど今

  : したほうがよい

  : あなたは何がVすると思うか
change : 不可算名詞
不可算名詞は単数扱い
in the morning , 過去の一点
should have done
  : すべきだったのに

Lesson5 関係詞

  : 都合がよい

Lesson6 仮定法
providing<provided> that S 原形
  : SVならば
I would rather
  = I wish
I wish S did
  : SがVすればよいのに
I wish S had done
  : SがVしていたらよかったのに
It is about/high time S did
  : そろそろ~する/もう~しても　いい時間だ
It is essantial/desirable/important that S 原形
  : Sが原形することは 不可欠/望ましい/重要 だ
propose/suggest that S <should> 原形
  : Vすることを提案する
If it were not for ~, would do = Were it not for ~, would do
  : (現在) ~がないならば
If it had not been for ~, would do = Had it not been for ~, would have done
  : (過去) ~がなかったならば

Lesson7 比較・形容詞・副詞
far/much/even/still 比較級
  : はるかに~
no less 形容詞/副詞 (ex: his family is no less important)
  : 同じように~である
much , little は形容詞を修飾できない
all the 比較級 (ex: He got all the more excited)
  : ますます~
not so much A as B
  = more B than A , B rather than A
  : AというよりむしろB
none the less for ~ (ex: loved his subordinates none the less for their faults.)
  : ~だがやはり
figure(数字),number,income,audienceの大小
  : large,smallを用いる
almost all of
  = most of
wrap up O
  : を仕上げる
the 比較級1 S1 V1, the 比較級2 S2 V2 (ex: the better a negotiator he is, the more successful ..)
  : ~1にS1がV1するほど、~2にS2はV2する
no more than ~
  : ~しか、単なる~
so be/助動詞 S -倒置- (ex: so is his secretary)
  : Sもまたそうする
such (a) 形容詞 名詞 that ~ (ex: such bad shape .. that ~)
  : とても...なので~

some (形容詞)
  : 可算名詞でも不可算名詞でも使える ex:some toast

# 文法メモ
〇仮定法
・仮定法未来(should would do)
　If S should V, S would[will] do
　If S were to V, S would do
・仮定法過去(did would do)
　If S did, S would do
・仮定法過去完了(had done woud have done)
　If S had done, S would have done
